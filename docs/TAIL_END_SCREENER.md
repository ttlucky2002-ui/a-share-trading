# 尾盘潜伏选股策略

## 动机

当前 `fundamental.py` 的中长期选股基于当日开盘后行情快照进行筛选，此时：
- 昨日利好已被消化，入选股可能已在当日高开，存在「抬轿子」风险。
- 缺乏对当日盘中资金行为的实时验证。

本模块将选股时机移至尾盘（14:30），采用「昨日初筛 + 今日盘中验证 + 尾盘精选」的三阶段流程，在第 T 日尾盘选出标的，博取 T+1 日溢价。

## 流程概览

```
时间           阶段              产出
─────────────────────────────────────────────────
14:00-14:10    Stage 1: 昨日池    基于昨日收盘的技术面/量能 → 初筛 200 只
14:10-14:25    Stage 2: 今日验证   分时趋势 + 资金流 + 量价 → 打分排序
14:25-14:30    Stage 3: 尾盘精选   行业分散 + 综合排名 → Top 10-15
```

## Stage 1: 昨日初筛

### 数据源
- `DataFeed.get_kline(code, count=80)` → 昨日 K 线 + 技术指标
- `DataFeed.get_stock_list()` → 全市场快照（市值、价格、板块）

实现上会显式剔除日期为今天的未完成日K，Stage 1 只使用最近一个完整交易日及其前一交易日计算涨跌幅、量比和技术结构；全市场快照只用于基础可交易边界，不用当日成交额或涨幅决定昨日候选池排序。

### 筛选条件

| 条件 | 阈值 | 目的 |
|------|------|------|
| 市值 | ≥ 30 亿 | 排除微盘流动性风险 |
| 价格 | 5-100 元 | 散户友好区间 |
| ST / 北交所 | 排除 | 风险过滤 |
| 昨日成交量 | > 5日均量 × 0.8 | 量能活跃，有资金关注 |
| 昨日收盘 vs MA5/MA10 | 收盘 > MA5 且 > MA10 | 短期趋势向上 |
| MACD | DIF > DEA | 动量偏多 |
| RSI | 35-65 | 未超买也不弱势 |
| 涨跌幅 | -5%~+5% | 排除异常波动 |

### 评分项

| 因子 | 权重 | 评分逻辑 |
|------|:----:|----------|
| 量比 | 25% | 昨日量/5日均量，clamp(ratio×25, 0-25) |
| 均线结构 | 25% | 站上MA5+MA10=15，MA20=10，MA60=10，金叉加成 |
| MACD | 20% | DIF>DEA=15，DIF>0=10 |
| 价格位置 | 15% | 昨日收盘在近10日区间60%-90%分位 |
| RSI | 15% | 40-60 得满分，30-40或60-70 减半 |

取评分前 200 只进入 Stage 2。

## Stage 2: 今日盘中验证

### 数据源
- `DataFeed.get_realtime_quotes()` → 今日实时行情
- `DataFeed.get_intraday_minute(code)` → 今日分时分钟数据
- `DataFeed.get_stock_list()` → 今日主力资金净流入
- `DataFeed.get_intraday_stock_fund_flow()` → 今日个股资金流向明细
- `DataFeed.get_sector_fund_flow()` / `get_concept_fund_flow()` → 板块资金

### 筛选条件

| 条件 | 阈值 | 目的 |
|------|------|------|
| 今日涨幅 | 0%~+5% | 无利空，也没一步到位 |
| 今日量比 | 0.8~2.5 | 活跃但未爆量 |
| 主力净流入 | > 0 | 资金认可 |
| 主力净流入占比 | 0%~10% | 有资金确认，但不过度追逐已拥挤标的 |
| 分时趋势 | 尾盘不跳水 | 资金维持 |
| 资金板块匹配 | 优先匹配当日资金领先行业/概念 | 寻找可能延续到 T+1 的主线 |

### 评分项

| 因子 | 权重 | 评分逻辑 |
|------|:----:|----------|
| 量价配合 | 30% | 今日量比×10 + 涨幅在0-5%内线性得分 + 当前价在今日区间上2/3 |
| 资金流入 | 30% | 主力净流入正值得分 + 超大单/大单占比得分 |
| 分时趋势 | 25% | 上涨分钟占比 + 尾盘30分钟趋势方向 + 价格在均线上方时间占比 |
| 板块资金 | 15% | 所在行业/概念板块当日资金流入排名得分 |

## Stage 3: 尾盘精选

### 综合分计算

```
综合分 = Stage1 分 × 30% + Stage2 分 × 70%
```

权重向今日验证倾斜，以当日资金行为为主要决策依据。

### 约束条件

- 同一行业板块不超过 2 只
- 同一概念板块不超过 2 只
- 排除当日涨停、跌停股

## 风险说明

- 本策略基于尾盘买入、T+1 卖出预期，不保证次日一定上涨。
- 盘中资金流数据为实时估算，与盘后官方数据可能存在差异。
- 尾盘买入后次日开盘存在低开风险。
- 不构成投资建议，仅供参考研究。
